# Juice Joint Website

### Web Communication Project

**Anushka Bhutani**

---

## Approach
**Start with flow, not visuals**

- Think from user’s perspective  
- Decide journey before design  
- Avoid unnecessary features  

---

## User Flow

- Landing page  
- Content sections  
- Call-to-action buttons  
- Footer & links  

Clear path → clear message

---

## Tech Choices

- HTML → structure  
- CSS → layout & colours  
- JavaScript → interaction  
- Bootstrap → responsiveness  

---

## Design Logic
**Visual decisions**

- Green → freshness  
- Yellow → energy  
- Red → fruit & attention  

---

## Tools & Libraries

- Bootstrap via CDN  
- Font Awesome for icons  
- Markdown for pitch deck  

---

## Lessons Learnt

- Simpler layouts work better  
- Structure matters more than effects  
- Small interactions go a long way  

---

## Favourite Part of Web Communication

I enjoyed working on **user experience** the most i.e.
how users move, click, and interact with the site.

---

## Final Takeaway

- Web communication is intentional  
- Every choice communicates  
- Simplicity is effective
